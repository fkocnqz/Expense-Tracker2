# Expense Tracker 

## Автор
Хайретдинов Руслан Ринатович

## Описание
Графическое приложение для учёта личных расходов. Позволяет добавлять транзакции, фильтровать их по категориям и датам, автоматически подсчитывать сумму, загружать курс доллара через публичный API и генерировать тестовые данные с помощью `random`. Все данные сохраняются в формате JSON.

## Стек технологий
- Python 3.8+
- `tkinter` (GUI)
- `json` (сохранение/загрузка)
- `urllib` (HTTP API запросы)
- `random` (генерация ID и тестовых данных)
- `unittest` (автоматическое тестирование)

## Как запустить
1. Убедитесь, что установлен Python 3.8+
2. Клонируйте репозиторий: `git clone <URL>`
3. Перейдите в папку: `cd expense_tracker`
4. Запустите приложение: `python main.py`
5. Запустите тесты: `python -m unittest test_expenses.py -v`

## Инструкция по использованию
| Действие | Как выполнить |
|----------|---------------|
| Добавить расход | Введите сумму > 0, выберите категорию, укажите дату `YYYY-MM-DD`, нажмите "Добавить" |
| Отфильтровать | Выберите категорию или "Все", задайте период "С" и "По", нажмите "Применить фильтр" |
| Посмотреть курс | Нажмите "Загрузить курс" (требуется интернет) |
| Сгенерировать тесты | Нажмите "Генератор тестов" для добавления 3-7 случайных записей |
| Сохранение | Автоматическое в `expenses.json` при каждом изменении |
